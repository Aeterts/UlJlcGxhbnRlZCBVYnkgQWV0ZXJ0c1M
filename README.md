# PvZ: Replanted Localization (Локализация)

Руссификатор полностью создан одним игроком из сообщества PvZ - Aeterts.

Весь текст и все текстуры переведены. Даже шрифты были доработаны вручную, чтобы лучше соответствовать стилю PvZ.

## Installation (Установка)
### 1. Автоматическая
  1) Скачать pvz-replanted-localizer.exe.
  2) Выбрать папку с игрой.
  3) Выбрать нужную версию и язык.
  4) Нажать на кнопку "Установить".
  5) Ждать.
  6) Запустить игру.
### 2. Ручная.
  1) Скачать архив из релиза для скачанной версии игры.
  2) Открыть папку с игрой (пример Steam): **...\Steam\steamapps\common\PVZ Replanted\Replanted_Data\StreamingAssets\aa**
  3) В эту папаку из архива перенести с заменой файл:
     * catalog.bin
  5) В папку **...\Replanted_Data\StreamingAssets\aa\StandaloneWindows64** из архива перенести с заменой файлы:
     * backgroundatlases_assets_all_ad9d2612bb65f45f86f9d103b8edf797.bundle
     * fonts_assets_all.bundle
     * gameplayatlases_assets_all.bundle
     * splashscreen_assets_all_3e8e7c96f34cba7a686bfda4792b9c61.bundle
     * uiatlasalmanac_assets_all.bundle
     * uiatlasshared_assets_all.bundle
     * localization_pc_assets_all_8118c7ddddd101ed68887c963273bcbd.bundle
  6) В папку **...\Replanted_Data\StreamingAssets\aa\StandaloneWindows64\defaultlocalgroup_assets_ui** из архива перенести с заменой файл:
     * sprites_83840c40f06744c3e2bf3db10c4a5d5a.bundle

## Uninstallation (Удаление)
  ### 1. Сделать восстановление игры в Steam или EA.

  ### 2. При автоматической установке.
  1) Запустить pvz-replanted-localizer.exe
  2) выбрать папку с игрой
  3) нажать на кнопку "Удалить".
